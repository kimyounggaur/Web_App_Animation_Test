SVG(Scalable Vector Graphics) 애니메이션은 해상도에 상관없이 이미지가 깨지지 않고, 용량이 작으며, 코드로 직접 제어할 수 있어 현대 웹 디자인에서 필수적인 요소입니다.

요청하신 대로 SVG 애니메이션의 주요 종류와 실제 적용 사례를 구체적으로 설명해 드리고, 직접 속성을 변경하며 시각적으로 원리를 이해할 수 있는 **인터랙티브 와이어프레임**을 하단에 준비했습니다. (참고로, 직접적인 영상 파일 삽입은 지원하지 않지만, 하단의 와이어프레임이 실제 영상처럼 살아 움직이는 실습 환경을 제공합니다.)

---

### 1. SVG 애니메이션 구현 방식의 종류

SVG를 애니메이션화하는 방법은 크게 세 가지로 나눌 수 있습니다.

#### A. CSS 애니메이션 (가장 권장됨)

SVG 내부의 요소(`<path>`, `<circle>`, `<rect>` 등)에 직접 CSS 클래스를 부여하여 `@keyframes`나 `transition`을 적용하는 방식입니다.

* **장점:** 웹 개발자에게 익숙하며, 성능이 좋고 가볍습니다.
* **한계:** 복잡한 경로 변형(Morphing)이나 개별 노드의 세밀한 제어는 어렵습니다.

#### B. SMIL (Synchronized Multimedia Integration Language)

SVG 태그 내부에 `<animate>`, `<animateTransform>`, `<animateMotion>` 등의 애니메이션 전용 태그를 직접 삽입하는 고전적인 방식입니다.

* **장점:** 외부 CSS나 JS 없이 SVG 파일 하나만으로 애니메이션이 완결됩니다. 요소를 특정 경로(Path)를 따라 움직이게 하는 데 탁월합니다.
* **한계:** 일부 모던 브라우저에서 지원을 축소하는 추세였으나 현재는 유지되고 있습니다. 코드가 복잡해질 수 있습니다.

#### C. JavaScript 라이브러리 (GSAP, Anime.js, Lottie 등)

자바스크립트를 이용해 SVG의 DOM을 직접 조작하거나, Lottie처럼 애프터이펙트 애니메이션을 JSON으로 추출해 렌더링하는 방식입니다.

* **장점:** 모양이 완전히 다른 형태로 변하는 모핑(Morphing), 스크롤 위치에 따른 애니메이션 등 매우 복잡하고 화려한 효과가 가능합니다.
* **한계:** 라이브러리 로드로 인해 페이지 용량이 약간 증가할 수 있습니다.

### 2. SVG 애니메이션의 실제 적용 사례 및 코드

가장 대표적이고 실무에서 많이 쓰이는 세 가지 사례입니다.

#### ① 선 그리기 애니메이션 (Line Drawing)

로고나 아이콘이 마치 펜으로 스케치하듯 그려지는 효과입니다. `stroke-dasharray`와 `stroke-dashoffset` 속성을 활용합니다.

* **적용 사례:** 웹사이트 첫 로딩 화면의 브랜드 로고, 서명 애니메이션, 스크롤 시 나타나는 일러스트.
* **핵심 원리:** 선을 점선으로 만들고(dasharray), 점선의 간격을 선의 전체 길이만큼 늘린 뒤, 오프셋(offset)을 0으로 줄이면서 선이 나타나게 합니다.

```css
.draw-line {
  stroke-dasharray: 1000; /* 선의 전체 길이 (예시) */
  stroke-dashoffset: 1000;
  animation: draw 2s ease-in-out forwards;
}

@keyframes draw {
  to { stroke-dashoffset: 0; }
}

```

#### ② 마이크로 인터랙션 (Micro-interactions)

사용자의 액션(호버, 클릭)에 반응하는 작고 섬세한 애니메이션입니다.

* **적용 사례:** 좋아요(하트) 버튼 클릭 시 하트가 통통 튀며 채워지는 효과, 햄버거 메뉴(≡)가 X자로 부드럽게 변하는 토글 버튼.
* **핵심 원리:** CSS의 `transform` (scale, rotate)과 `fill` (색상 채우기) 속성을 조합합니다.

#### ③ 데이터 시각화 (Data Visualization)

차트나 그래프의 수치가 변경될 때 부드럽게 크기나 모양이 변하는 효과입니다.

* **적용 사례:** 대시보드의 도넛 차트가 차오르는 효과, 막대 그래프가 아래에서 위로 자라나는 효과.
