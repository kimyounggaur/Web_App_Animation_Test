CSS 애니메이션은 웹 페이지에 생동감을 불어넣고 사용자 경험(UX)을 크게 향상시키는 강력한 도구입니다. 요청하신 대로 CSS 애니메이션의 핵심 종류와 구체적인 적용 방법을 설명해 드리고, 직접 속성을 조작하며 시각적으로 이해할 수 있는 **인터랙티브 와이어프레임**을 하단에 준비했습니다.

---

### 1. CSS 애니메이션의 두 가지 주요 종류

CSS로 요소를 움직이게 하는 방법은 크게 트랜지션(Transition)과 **키프레임(Keyframes)** 두 가지로 나뉩니다.

#### A. 트랜지션 (Transitions)

가장 단순한 형태의 애니메이션입니다. 주로 마우스 오버(`:hover`)나 클릭(`:active`) 같은 '상태 변화'가 일어날 때, A 상태에서 B 상태로 부드럽게 넘어가는 효과를 줍니다.

* **특징:** 시작점과 끝점만 존재하며, 중간 단계를 세밀하게 제어할 수는 없습니다.
* **실제 적용 방법:**
```css
.button {
  background-color: white;
  /* 배경색 변경을 0.3초 동안, 부드럽게(ease) 진행 */
  transition: background-color 0.3s ease; 
}
.button:hover {
  background-color: black; /* 마우스를 올리면 검은색으로 변경 */
}

```



#### B. 키프레임 애니메이션 (@keyframes)

상태 변화 없이도 페이지 로드와 함께 스스로 움직이거나, 복잡하고 다단계적인(multi-step) 움직임을 만들 때 사용합니다.

* **특징:** `0%`(시작)부터 `100%`(끝) 사이의 중간 프레임(예: `50%`)을 개발자가 마음대로 설정할 수 있습니다.
* **실제 적용 방법:**
```css
/* 1. 애니메이션 시퀀스 정의 */
@keyframes slideAndFade {
  0% { transform: translateX(0); opacity: 1; }
  50% { transform: translateX(50px); opacity: 0.5; }
  100% { transform: translateX(100px); opacity: 0; }
}

/* 2. 요소에 애니메이션 적용 */
.box {
  /* 이름, 지속시간, 속도곡선, 반복횟수 */
  animation: slideAndFade 2s ease-in-out infinite; 
}

```

### 2. 애니메이션의 디테일을 결정하는 핵심 속성들

애니메이션의 느낌을 결정하는 것은 '속도 곡선'과 '시간'입니다.

* **`duration` (지속 시간):** 애니메이션이 완료되는 데 걸리는 시간입니다. (예: `0.5s`, `200ms`)
* **`timing-function` (속도 곡선):** 애니메이션의 가속과 감속을 제어합니다.
* `linear`: 일정한 속도로 움직입니다.
* `ease`: 시작과 끝은 부드럽고 중간에 빨라집니다. (기본값)
* `ease-in`: 천천히 시작해서 점점 빨라집니다.
* `ease-out`: 빠르게 시작해서 천천히 끝납니다.


* **`delay` (지연 시간):** 애니메이션이 시작되기 전 대기하는 시간입니다.
* **`iteration-count` (반복 횟수):** 몇 번 반복할지 결정합니다. 계속 반복하려면 `infinite`를 사용합니다.
